# Component with Attributes

## Definition

```js
{
  _style: {
    entity: 'align=left;overflow=fill;html=1;dropTarget=0;whiteSpace=wrap;',
  },
  _width: 0,
  _height: 90,
}
```

## Usage

```js
import { ComponentWithAttributes } from '@dinghy/standard-components-diagrams/uml'

<ComponentWithAttributes/>
```
