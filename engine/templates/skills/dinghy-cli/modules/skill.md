# skill

Install or update Dinghy skills (Claude Code slash commands).

## Usage

```sh
$ dinghy skill [options]
```

## Options

| Name                    | Description                                                                                                      |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------- |
| `--skills-path=<value>` | The target folder to install skills into. Auto-detected from .claude/skills or ~/.claude/skills if not provided. |
| `-q, --quiet`           | Quiet mode                                                                                                       |
