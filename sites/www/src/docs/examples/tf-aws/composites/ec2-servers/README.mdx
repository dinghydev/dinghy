## Connecting to your EC2 Server

After your EC2 server(s) have been provisioned, you can easily manage and connect to them:

1. Use [dinghy aws list](https://dinghy.dev/references/commands/engine/aws/list) to display all EC2 instances.
2. Use [dinghy aws connect](https://dinghy.dev/references/commands/engine/aws/connect) to start an interactive connection to any instance, with no extra setup required.
