# devcontainer

Start the project in devcontainer

## Usage
```sh
$ dinghy devcontainer [options]
$ dinghy dc [options]
```
## Options
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `--no-open` | Do not open the devcontainer. Generate .devcontainer.json only if not exist  |
| `--volumns` | Additional volumns to mount to the devcontainer  |
| `--workspace` | The workspace folder to use in the devcontainer  |
