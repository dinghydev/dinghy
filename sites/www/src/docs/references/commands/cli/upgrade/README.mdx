# upgrade

Upgrade Dinghy Cli to latest or specified version

## Usage
```sh
$ dinghy upgrade [options]
$ dinghy up [options]
```
## Options
| Name            | Description           | Default                   |
| ---------------- | --------------- | ----------------------------- |
| `--[no]-update-project` | By default, upgrade will also update your project engine version if `dinghy.config.yaml` exist in current folder. You may pass `--no-update-project` to skip it. | `true`  |
| `-v, --version` | The version to upgrade to | `latest`  |
