# Engine
Dinghy Engine which runs commands inside a Docker container.
## Usage
```sh
$ dinghy [command] [options]
```

## Options
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `-v, --version` |Show version  |
| `[--commands=<value>]` |Commands to run in sequence, parallel commands split by  \| from value  |
## Global Options
| Name            | Description           | Default                   |
| ---------------- | --------------- | ----------------------------- |
| `-h, --help` |Show help |  |
| `--app-home=<value>` |The path to the Dinghy app home directory `APP_HOME`|  |
| `--debug` |Enable debug mode |  |
| `--output=<value>` |The path to the output directory | `output` |
