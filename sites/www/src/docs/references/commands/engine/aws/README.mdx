# aws
Run aws command with awscli image
## Usage
```sh
$ dinghy aws [command] [options]
```
## Commands
- [aws bash](./bash/README.mdx)
- [aws connect](./connect/README.mdx)
- [aws list](./list/README.mdx)
## Example usage

To execute `aws s3 ls`, run:
```sh
dinghy aws s3 ls
```
