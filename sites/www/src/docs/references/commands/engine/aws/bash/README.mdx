# aws bash

Run bash command with awscli image

## Usage
```sh
$ dinghy aws bash [STACK] [options]
```
## Arguments
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `STACK` | Stack name  |
## Options
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `--awscli-options` | Additional awscli options  |
| `--debug` |   |
