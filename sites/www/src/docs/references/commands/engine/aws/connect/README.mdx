# aws connect
Connect to a specific instance in the stack
## Usage
```sh
$ dinghy aws connect [STACK] [options]
```
## Arguments
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `stack` |Stack name  |
## Options
| Name            | Description           | Default                   |
| ---------------- | --------------- | ----------------------------- |
| `-i, --instance-id <value>` |The instance id to connect to, could be sequence or actual instance id | `1` |
