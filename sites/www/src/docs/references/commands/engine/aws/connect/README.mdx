# aws connect

Connect to a specific instance in the stack

## Usage
```sh
$ dinghy aws connect [STACK] [options]
```
## Arguments
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `STACK` | Stack name  |
## Options
| Name            | Description           | Default                   |
| ---------------- | --------------- | ----------------------------- |
| `--awscli-options` | Additional awscli options |   |
| `--debug` |  |   |
| `-i, --instance-id` | The instance id to connect to, could be sequence or actual instance id | `1`  |
