# aws list

List connectable instances in the stack

## Usage
```sh
$ dinghy aws list [STACK] [options]
```
## Arguments
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `STACK` | Stack name  |
## Options
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `--awscli-options` | Additional awscli options  |
| `--debug` |   |
