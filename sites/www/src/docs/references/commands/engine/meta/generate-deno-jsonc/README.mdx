# meta generate-deno-jsonc
Generate deno.jsonc with npm imports from dinghy.config.yml packages.
## Usage
```sh
$ dinghy meta generate-deno-jsonc JSONCFILE [options]
```
## Arguments
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `jsoncFile` |Target deno.jsonc file path to generate  |
