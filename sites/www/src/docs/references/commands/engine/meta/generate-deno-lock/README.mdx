# meta generate-deno-lock
Generate deno.lock by extracting npm entries from the engine lock file.
## Usage
```sh
$ dinghy meta generate-deno-lock LOCKFILE [options]
```
## Arguments
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `lockFile` |Target deno.lock file path to generate  |
