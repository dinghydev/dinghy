# meta override
Override a Dinghy file by copying it into the project.
## Usage
```sh
$ dinghy meta override PATH [options]
```
## Arguments
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `path` |Dinghy file path to override  |
