# slide serve
Serve built slides
## Usage
```sh
$ dinghy slide serve [options]
```
## Options
| Name            | Description           | Default                   |
| ---------------- | --------------- | ----------------------------- |
| `--port=<value>` |Port to listen on | `3000` |
