# slide start
Start slides in development mode
## Usage
```sh
$ dinghy slide start [options]
```
## Options
| Name            | Description           | Default                   |
| ---------------- | --------------- | ----------------------------- |
| `--port=<value>` |Port to listen on | `3000` |
