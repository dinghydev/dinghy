# tf bash

Run bash command with tf image

## Usage
```sh
$ dinghy tf bash [STACK] [options]
```
## Arguments
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `STACK` | Stack name  |
## Options
| Name            | Description           | Default                   |
| ---------------- | --------------- | ----------------------------- |
| `--debug` |  |   |
| `--diff-changes-max-lines` | Max lines of diff changes | `10`  |
| `--lock` | Lock the state during plan |   |
| `--plan-file` | Plan file name | `tf.plan`  |
| `--tf-options` | Additional tf options |   |
