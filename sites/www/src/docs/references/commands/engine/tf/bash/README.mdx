# tf bash
Run bash/tofu/terraform with tf image on selected stack
## Usage
```sh
$ dinghy tf bash [STACK] [options]
$ dinghy tf tofu [STACK] [options]
$ dinghy tf terraform [STACK] [options]
```
## Arguments
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `stack` |Stack name  |
