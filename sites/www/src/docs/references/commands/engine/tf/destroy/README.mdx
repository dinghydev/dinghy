# tf destroy

Destroy all managed resources

## Usage
```sh
$ dinghy tf destroy [STACK] [options]
```
## Arguments
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `STACK` | Stack name  |
## Options
| Name            | Description           | Default                   |
| ---------------- | --------------- | ----------------------------- |
| `--auto-approve` | Auto approve the destroy operation |   |
| `--debug` |  |   |
| `--diff-changes-max-lines` |  | `10`  |
| `--tf-options` | Additional tf options |   |
