# tf destroy
Run `terraform destroy` command for selected stacks
## Usage
```sh
$ dinghy tf destroy [STACK] [options]
```
## Arguments
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `stack` |Stack name  |
## Options
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `-a, --auto-approve` |Auto approve the destroy operation  |
