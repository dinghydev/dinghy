# tf diff

Show a preview of actions in selected targets

## Usage
```sh
$ dinghy tf diff [STACK] [options]
```
## Arguments
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `STACK` | Stack name  |
## Options
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `--debug` |   |
| `--stage` | Stages to work with  |
