# tf diff

Show a preview of actions in selected targets. It combines render, init and plan commands. You may provide any addtiional arguments which are valid for those commands.

## Usage
```sh
$ dinghy tf diff [STACK] [options]
```
## Arguments
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `STACK` | Stack name  |
