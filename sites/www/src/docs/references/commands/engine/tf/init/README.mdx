# tf init

Initialize working directory

## Usage
```sh
$ dinghy tf init [STACK] [options]
```
## Arguments
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `STACK` | Stack name  |
## Options
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `--auto-create-backend` | Auto create backend bucket if it doesn't exist  |
| `--debug` |   |
| `--enable-additional-providers` | Enable additional providers which not installed by default  |
| `--tf-options` | Additional tf options  |
