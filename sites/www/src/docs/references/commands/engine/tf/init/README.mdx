# tf init
Run `terraform init` command for selected stacks
## Usage
```sh
$ dinghy tf init [STACK] [options]
```
## Arguments
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `stack` |Stack name  |
## Options
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `--auto-create-backend` |Auto create backend bucket if it doesn't exist `DINGHY_TF_INIT_AUTO_CREATE_BACKEND` |
