# tf plan
Run `terraform plan` command for selected stacks
## Usage
```sh
$ dinghy tf plan [STACK] [options]
```
## Arguments
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `stack` |Stack name  |
## Options
| Name            | Description           | Default                   |
| ---------------- | --------------- | ----------------------------- |
| `--diff-changes-max-lines=<value>` |Max lines of diff changes | `10` |
| `--lock` |Lock the state during plan | `false` |
