# tf up

Create or update the resources in selected targets

## Usage
```sh
$ dinghy tf up [STACK] [options]
```
## Arguments
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `STACK` | Stack name  |
## Options
| Name            | Description                     |
| ---------------- | ----------------------------- |
| `--debug` |   |
| `--stage` | Stages to work with  |
