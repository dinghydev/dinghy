# upgrade

Upgrade/lock project Dinghy version, to the latest version by default

## Usage
```sh
$ dinghy upgrade [options]
$ dinghy up [options]
```
## Options
| Name            | Description           | Default                   |
| ---------------- | --------------- | ----------------------------- |
| `-v, --version` | The version to upgrade to | `latest`  |
