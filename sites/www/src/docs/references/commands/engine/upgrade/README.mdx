# upgrade
Upgrade/lock project Dinghy version.
## Usage
```sh
$ dinghy upgrade [options]
$ dinghy up [options]
```
## Options
| Name            | Description           | Default                   |
| ---------------- | --------------- | ----------------------------- |
| `-v, --version=<value>` |The version to upgrade to | `latest` |
